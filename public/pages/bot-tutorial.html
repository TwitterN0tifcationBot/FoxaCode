Creating a Discord Bot with JavaScript

In this tutorial, we will create a simple Discord bot using JavaScript and the Discord.js library.

Step 1: Setting Up

First, you need to have Node.js installed on your computer. You can download it from nodejs.org.

Step 2: Creating a New Discord Application

  1. Go to the Discord Developer Portal.
  2. Click on "New Application" and give it a name.
  3. Navigate to the "Bot" tab and click "Add Bot".
  4. Copy the bot token. You will need this later.

Step 3: Setting Up Your Project


// Create a new directory for your bot
mkdir my-discord-bot
cd my-discord-bot

// Initialize a new Node.js project
npm init -y

// Install the Discord.js library
npm install discord.js

Step 4: Writing the Bot Code

Create a file named index.js and add the following code:


const { Client, GatewayIntentBits } = require('discord.js');
const client = new Client({ intents: [GatewayIntentBits.Guilds, GatewayIntentBits.GuildMessages, GatewayIntentBits.MessageContent] });

const token = 'YOUR_BOT_TOKEN_HERE';

client.once('ready', () => {
    console.log('Bot is online!');
});

client.on('messageCreate', message => {
    if (message.content === '!ping') {
        message.channel.send('Pong!');
    }
});

client.login(token);

Step 5: Running Your Bot

To start your bot, run the following command in your project directory:


node index.js

Your bot should now be online and respond to the !ping command in your Discord server!
